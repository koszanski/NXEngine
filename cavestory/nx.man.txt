<Cave Story for RetroFW, Based on NXEngine 1.0.0.4.>

profile.dat and settings file is located in $HOME/.cavestory/

Default controls:

Directional movement - D-PAD
Jump - B 
Fire - Y 
Weapon Previous - A
Weapon Next - X
Inventory - L
Map - R
Select - Menu
Start - Options
